# System Goal (Different Perspectives)

---

# Final System Goal

The system is a stateful conversational decision engine that builds an evolving understanding of a user's fundraising intent and uses that understanding to guide product selections and referral actions that increase engagement, purchases, and school donations.

---

# Architecture-Level Goal

The system exists to:

Transform conversation into structured decision context that drives user behavior toward fundraising outcomes.

This captures:

- conversation → signal extraction  
- signals → decision context  
- context → reasoning  
- reasoning → recommendations  
- recommendations → behavior  
- behavior → fundraising  

---

# System Goal (Minimal — Strong Version)

Conversationally guide users toward decisions that maximize school fundraising.

---

# System Goal (Engineering Version)

Design a conversation-aware AI service that:

- accumulates user decision signals  
- models fundraising intent  
- reasons over domain data  
- generates grounded recommendations  
- encourages referral participation  
- maintains multi-turn context
