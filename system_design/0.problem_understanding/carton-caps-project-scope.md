# Cartoon Caps Project Scope

# The Challenge — AI-Powered Chat Agent for Carton Caps

Carton Caps is an app that empowers consumers to raise money for schools while buying everyday products.

The app is already full-featured and publicly available.

The Carton Caps team is exploring how AI can:

- enhance user engagement
- support smarter decision-making

Goal:

Prototype an AI-driven conversational assistant that:

- provides personalized product recommendations
- explains referral process
- improves user engagement
